## General Info
This browser based web application to allow users to rate and observe ratings about a list of parks in Burnaby.

	
## Technologies
Technologies used for this project:
* HTML, CSS
* JavaScript
* Bootstrap 
* Firebase
* Font awesome
* Gibhub
